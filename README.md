# This repository contains the labs for the Web Technologies course

## Web Technologies Labs

- [Lab 2](lab2.pdf) : **Django Setup & Git Essentials** – Focused on environment configuration (venv), Django installation, and version control using Git and GitHub.
- [Lab 3](lab3.pdf) : **MVT Architecture & URL Routing** – Explored the Model-View-Template pattern, mapping URLs to views, and rendering basic dynamic content.
- [Lab 4](lab4.pdf) : **Advanced Templates & Layouts** – Implemented Template Inheritance, static file management (images/icons), and styled components using CSS.
